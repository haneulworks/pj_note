CSS

주석/빈줄

  • 종료 주석은 작성하지 않는다.
  • 코드 그룹 간 1줄의 빈 줄을 허용한다. 단, 1줄을 초과하지 않는다.
1
2
3
4
5
/* title */
h1 { font-size: 1.2rem; }
 
/* description */
p { font-size: 0.9rem; }
cs

세미클론(;)

  • 마지막으로 사용된 속성에도 세미클론(;)을 사용한다.
1
h1 { margin:0; padding:0; }
